Next we will learn how to use forms to make our website interactive with form tags!

  • First attribute/tag inside the form tag is the input tag which makes a box where the user can type in, it's self closing. The input tag can have various specifications like type ="text", now its clear you want a text input. Can use the placeholder="" attribute to hint at the user to what they are supposed to put in the input box (looks faded and dissappears as soon as the user types)
  • The next is a checbox attribute which is exactly what is sounds like.You might want to have the checkbox automatically checked, you can do that with just typing the word checked inside the input tag
  • The radio attribute is like the checkbox attribute except you can have multiple radio boxes and you can only check off one. In order to only be able to pick one you have to link them so the program knows they are part of a set where you can only pick one out of all of them. You link them up by giving them a name. You can also use the value attribute to label the choices with a specific value which then you can store. ex: value="lovedogs" or value="hatedogs". Default choice, put the word selected in the opening option tag of the choice that you want to be the default choice
  • The select attribute creates a drop down where you can pick from the choices given. To put choices in the drop down you use the options tag within the opening and closing of the select tag
  • The last is the submit button which is a type of input. You can customize the submit button to say different thing by using value=""
